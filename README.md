# Roamly

AI-powered group travel planning. Helps a group converge on destinations everyone can agree on, then builds out the itinerary.

## Live

[roamly.vientapps.com](https://roamly.vientapps.com)

## Tech Stack

- Next.js 15
- TypeScript
- Supabase
- Anthropic Claude
- Stripe
- Tailwind CSS v4
- Cloudflare
